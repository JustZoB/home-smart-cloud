Ознакомиться с постановлением

Возможности Home Smart Cloud

Облачный сервис не ограничен по объёму места для хранения данных. Расходы за любой период с момента установки системы будут в постоянном доступе.

Установить систему при постройке или заменить старые счетчики очень легко. Устройство не имеет проводов, что делает его “чистым” для установки даже после окончания ремонта или в уже эксплуатируемом помещении.

Личный кабинет пользователя показывает полную статистику потребления всех видов коммунальных услуг. Функционал кабинета может быть расширен от частного использования пользователем до контролирования неограниченного количества счетчиков одновременно.

Что нужно для установки?

Счетчик импульсов

Компактные устройства, легко устанавливающиеся на импульсные счетчики любого поколения. Один нод контролирует показания сразу нескольких приборов учета. Устанавливать отдельное устройство на каждый счетчик не требуется.

Шлюз для сбора данных

Контроллеры передают показания на шлюз, установленный в любой удобной точке дома, будь то подвал или чердак. Радиус видимости шлюза легко охватит целый подъезд многоквартирного дома.

Точка доступа Wi-Fi или SiM-карта

Для обслуживания всех устройств достаточно всего одной SIM-карты или одной точки доступа Wi-Fi в шлюзе. Данные с него направляются в наш облачный сервис для дальнейшего анализа, расчета и отправки данных как в компании ТСЖ, так и конечным пользователям.

Как устроена система облачного хранения и передачи данных?

  • Соединение с облаком не требует дополнительного оборудования или сложной инфраструктуры.
  • Для обслуживания всех устройств достаточно SIM-карты или точки доступа Wi-Fi в шлюзе.
  • Цена трафика будет определяться вашим абонентским тарифом.

Существует три варианта устройства для пользователя:

  1. Wifi для частных и коттеджных поселков
  2. 3G для частного сектора
  3. LoraWAN для централизованной установки по всему дому/поселку
